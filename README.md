# Estatística Projeto TADS

Projeto desenvolvido na disciplina de **Estatística** do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas (TADS)**.

A aplicação tem como objetivo aplicar conceitos estatísticos utilizando programação, permitindo a análise e visualização de dados através de uma interface interativa desenvolvida com Streamlit.

## 🚀 Aplicação

Acesse a aplicação online:

https://estatisticaprojeto-tads2026.streamlit.app/

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
- Bibliotecas de análise e visualização de dados

## 📚 Disciplina

Projeto desenvolvido como atividade da disciplina de **Estatística - TADS**.
